Integration and Differentiation



Differentiation

For a single variable function \(f: [a,b] \rightarrow \mathbb{R}\), its derivative is defined as: \[\partial_x f(x) = \lim_{t \rightarrow x} \frac{f(t) - f(x)}{t-x} \]

Integration

For a single variable function \(f: [a,b] \rightarrow \mathbb{R}\), its Reimann integral from \(t \in [a,b]\) is defined as: \[\int_t^x f(x) dx = \lim_{\| P \| \rightarrow 0} \sum_1^n f(m_k) (x_{k+1} - x_k) \] where \(P\) is a tagged partition of \([t,x]\) with partition points \(x_1 ... x_{n+1}\) and tags \(m_1 ... m_n\).
